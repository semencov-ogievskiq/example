# example

## Описание
Этот проект является конечной сборкой всех моих тестовых проектов. 

На данный момент он содержит проекты -
1) server - http сервер и socket io сервер
2) admin_panel - Nuxt проект в виде Админской панели сайта

## Установка 
Выполните следующие команды
```bash
# сделайте клон репозитория
$ git clone https://github.com/semencov-ogievskiq/example.git

# Перейдите в каталог проекта и подгрузите зависимости
$ npm install
```
Далее выполните sql команды находящиеся в файле '/db.sql' на своем mysql сервере.
Укажите в '/server.config.js' данные подключения к вашему mysql серверу

И запустите проект выполнив команду в консоле
```bash
# Запуск проекта
$ npm run dev

```
Перейдите по адресу 'http://localhost/admin' и авторизуйтесь под пользователем 'test@mail.ru' 'testtest'
